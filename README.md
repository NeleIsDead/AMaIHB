Этот репозиторий создан чтобы собирать и хранить тексты докладов по истории, потому что я искренне сомневаюсь что кто-то хочет готовить по 5-6 докладов к каждой практике самостоятельно, так как даже с нейросетями их писать и редактировать очень очень лень - это лишняя работа. Поэтому собственно есть предложение, чтобы мы просто дружно, общими усилиями распределили все темы за семестр по людям, и написали по ним доклады, чтобы потом каждый человек перед занятием мог просто зайти, скачать себе нужные тексты, прочитать их и быть уверен, что на практике все будет нормально.

В этой таблице записаны все темы(вопросы) по истории. В идеале каждый человек должен выбрать по 4-6 тем и написать мне об этом в лс в телеграмме (@Nele_was_lost) (Создавать для этого pull реквесты звучит как лишняя работа). Я запишу вас в таблицу как выбравшего тему человека, чтобы эту тему больше никто не брал. Как только доделаете текст, можете либо скинуть файл мне либо самостоятельно создать pull реквест на репозиторий github.
Просьба в приоритет ставить доклады с меньшими номерам и не злоупотреблять этой идеей. Репозиторий существует чтобы нам всем было проще жить и меньше делать.
